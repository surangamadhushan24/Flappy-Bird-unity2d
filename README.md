# Flappy Bird Unity Clone

A simple Flappy Bird clone built with Unity, featuring a bird that navigates through pipes by flapping its wings.

## Features

- Classic Flappy Bird gameplay mechanics
- Procedural pipe generation
- Score tracking
- Simple, clean graphics
- Responsive controls

## Requirements

- Unity 2021.3 or later
- Windows, macOS, or Linux

## Installation and Setup

1. Clone this repository:
   ```
   git clone https://github.com/yourusername/flappy-bird-unity.git
   cd flappy-bird-unity
   ```

2. Open the project in Unity:
   - Launch Unity Hub
   - Click "Open" and select the cloned project folder

3. Open the main scene:
   - In the Project window, navigate to `Assets/Scenes/`
   - Double-click `SampleScene.unity` to open it

4. Play the game:
   - Press the Play button in the Unity Editor
   - Or build the project for your target platform

## How to Play

- Click the left mouse button or press the spacebar to make the bird flap its wings
- Navigate through the gaps between the pipes
- Avoid hitting the pipes or the ground
- Try to achieve the highest score possible!

## Controls

- **Mouse Click** or **Spacebar**: Flap wings

## Project Structure

- `Assets/BirdScript.cs`: Controls the bird's movement and physics
- `Assets/PipeMoveScript.cs`: Handles pipe movement
- `Assets/PipeSpawnerScript.cs`: Manages pipe spawning
- `Assets/Pipe.prefab`: Pipe prefab
- `Assets/Scenes/SampleScene.unity`: Main game scene
